# Directory Watcher

A Python script that monitors a folder and detects file changes.

## Description
This project continuously checks a directory and identifies files that are added or removed.

## Features
- Real-time monitoring (interval-based)
- Detects file additions
- Detects file deletions
- Simple and useful automation tool

## Technologies Used
- Python
- os module
- time module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python watcher.py

4. Enter folder path

## Example

Output:
File added: report.txt  
File removed: old_data.csv
